---
title: useSignInMagicLink
description: Send a magic-link sign-in email.
---

The session is established when the user clicks the emailed link, so no session-cache side effects run here.

## Usage

```tsx
import { authClient } from "@/lib/auth-client"
import { useSignInMagicLink } from "@better-auth-ui/react/plugins/magic-link"

const { mutate: signInMagicLink } = useSignInMagicLink(authClient)

signInMagicLink({
  email: "alice@example.com",
  callbackURL: "/dashboard"
})
```

`useSignInMagicLink` requires the Better Auth `magicLink` plugin. Import your configured `authClient`; it already carries the plugin methods from `createAuthClient`.

## Options factory

```tsx
import { signInMagicLinkOptions } from "@better-auth-ui/core/plugins/magic-link"
import { useMutation } from "@tanstack/react-query"

const { mutate } = useMutation(signInMagicLinkOptions(authClient))
```
