---
title: useDeleteOrganization
description: Delete an organization.
---

Organization mutations require `organizationClient()` on the Better Auth client.

## Usage

```tsx
import { useAuth, useDeleteOrganization } from "@better-auth-ui/react"

const { authClient } = useAuth()
const mutation = useDeleteOrganization(authClient)

mutation.mutate(/* params */)
```

## Options factory

```tsx
import { deleteOrganizationOptions } from "@better-auth-ui/react"
import { useMutation } from "@tanstack/react-query"

const mutation = useMutation(deleteOrganizationOptions(authClient))
```
