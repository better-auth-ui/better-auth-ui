---
title: useAcceptInvitation
description: Accept an organization invitation.
---

Organization mutations require `organizationClient()` on the Better Auth client.

## Usage

```tsx
import { useAuth, useAcceptInvitation } from "@better-auth-ui/react"

const { authClient } = useAuth()
const mutation = useAcceptInvitation(authClient)

mutation.mutate(/* params */)
```

## Options factory

```tsx
import { acceptInvitationOptions } from "@better-auth-ui/react"
import { useMutation } from "@tanstack/react-query"

const mutation = useMutation(acceptInvitationOptions(authClient))
```
