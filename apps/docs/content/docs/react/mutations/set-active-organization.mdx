---
title: useSetActiveOrganization
description: Set the active organization for the current session.
---

Organization mutations require `organizationClient()` on the Better Auth client.

## Usage

```tsx
import { useAuth, useSetActiveOrganization } from "@better-auth-ui/react"

const { authClient } = useAuth()
const mutation = useSetActiveOrganization(authClient)

mutation.mutate(/* params */)
```

## Options factory

```tsx
import { setActiveOrganizationOptions } from "@better-auth-ui/react"
import { useMutation } from "@tanstack/react-query"

const mutation = useMutation(setActiveOrganizationOptions(authClient))
```
