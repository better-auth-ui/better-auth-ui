## Usage

<Demo src="/demos/shadcn/emails/password-changed-email" className="h-[720px]" />

```tsx
import { PasswordChangedEmail } from "@better-auth-ui/react"

export function PasswordChangeNotificationEmail() {
  return (
    <PasswordChangedEmail
      email="user@example.com"
      timestamp="January 15, 2024 at 3:30 PM"
      appName="My App"
      supportEmail="support@example.com"
      secureAccountURL="https://example.com/secure-account"
      logoURL="https://example.com/logo.png"
      darkMode={true}
    />
  )
}
```

## Installation

```npm
npx shadcn@latest add https://better-auth-ui.com/r/password-changed-email.json
```

## Features

- Password change notification
- Timestamp of the change
- Security action button if unauthorized
- Support contact information
- Customizable branding and styling
- Support for light/dark mode themes
- Localization support
