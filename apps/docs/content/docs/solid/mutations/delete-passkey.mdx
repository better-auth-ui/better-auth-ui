---
title: useDeletePasskey
description: Delete a passkey from the current user's account.
---

Requires the Better Auth passkey plugin.

## Usage

```tsx
import { deletePasskeyOptions } from "@better-auth-ui/core/plugins/passkey"
import { useMutation } from "@tanstack/solid-query"

const deletePasskey = useMutation(() => deletePasskeyOptions(authClient))

deletePasskey.mutate({ id: "passkey-id" })
```
