---
title: revokeSessionOptions
description: Revoke one active session for the current user.
---

## Usage

```tsx
import { revokeSessionOptions } from "@better-auth-ui/core"
import { createMutation } from "@tanstack/solid-query"

const revokeSession = createMutation(() => revokeSessionOptions(authClient))

revokeSession.mutate({ token: "session-token" })
```

## Cache

Refetch `listSessions` after revocation.
