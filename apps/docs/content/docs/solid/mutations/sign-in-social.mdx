---
title: signInSocialOptions
description: Social sign-in mutation options.
---

Starts the provider OAuth flow and redirects through Better Auth.

## Options factory

```tsx
import { signInSocialOptions } from "@better-auth-ui/core"
import { createMutation } from "@tanstack/solid-query"

const signInSocial = createMutation(() => signInSocialOptions(authClient))

signInSocial.mutate({
  provider: "github",
  callbackURL: "/dashboard"
})
```
