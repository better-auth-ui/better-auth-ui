---
title: useUnlinkAccount
description: Unlink a social account from the current user.
---

## Usage

```tsx
import { unlinkAccountOptions } from "@better-auth-ui/core"
import { useMutation } from "@tanstack/solid-query"

const unlinkAccount = useMutation(() => unlinkAccountOptions(authClient))

unlinkAccount.mutate({ providerId: "github", accountId: "github-account-id" })
```

## Cache

Refetch linked accounts after unlinking.
