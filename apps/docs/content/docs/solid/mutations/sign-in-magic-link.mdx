---
title: useSignInMagicLink
description: Magic-link sign-in mutation hook.
---

Requires the Better Auth magic-link plugin. If TypeScript cannot infer the plugin methods from your client, type or cast `authClient` as `MagicLinkAuthClient` from `@better-auth-ui/core/plugins/magic-link`.

## Usage

```tsx
import { useSignInMagicLink } from "@better-auth-ui/solid"

const sendMagicLink = useSignInMagicLink(authClient)

sendMagicLink.mutate({
  email: "alice@example.com",
  callbackURL: "/dashboard"
})
```

## Options factory

```tsx
import { signInMagicLinkOptions } from "@better-auth-ui/core/plugins/magic-link"
import { createMutation } from "@tanstack/solid-query"

const sendMagicLink = createMutation(() => signInMagicLinkOptions(authClient))
```
