## Usage

<Demo src="/demos/heroui/email/email-verification-email" className="h-[620px]" />

```tsx
import { EmailVerificationEmail } from "@better-auth-ui/heroui"

export function VerificationEmail() {
  return (
    <EmailVerificationEmail
      url="https://example.com/verify?token=example-token"
      email="user@example.com"
      appName="My App"
      expirationMinutes={60}
      darkMode={true}
    />
  )
}
```

## Features

- Verification button and fallback URL
- Expiration time information
- Security notice for unauthorized requests
- Customizable branding and styling
- Support for light/dark mode themes
- Localization support
